#Quiz App

A simple and interactive quiz web application built using HTML, CSS, JavaScript.
This Quiz App allows users to answer multiple-choice questions and test their
knowledge ina fun and interactive way.It is designed as a beginner-friendly project
to practice core web development concepts.

##Features:
-Multiple-choice Questions
-Displays questions one by one
-Simple and user-friendly interface

##Technologies used:
-HTML
-CSS
-JavaScript
